# CSV to Dashboard

## Overview
This project converts any CSV dataset into an interactive dashboard. Users can upload a CSV file and instantly explore the data through charts, statistics, filters, and data cleaning tools.
The goal of this project is to make data exploration simple and interactive without writing additional code.

## Problem Statement
Raw datasets often contain missing values, duplicates, and large amounts of information that are difficult to analyze manually. The objective of this project is to provide an easy way to explore and understand datasets through an interactive dashboard.

## Tools and Technologies

Python
Streamlit
Pandas

## Project Workflow
1. Upload CSV dataset
2. Preview dataset
3. Display dataset information
4. Check missing values
5. Generate summary statistics
6. Perform data cleaning
7. Apply filters
8. Create visualizations
9. Extract key insights

## Features

CSV file upload
Dataset preview
Missing value analysis
Duplicate removal
Data filtering
Histogram
Box Plot
Scatter Plot
Bar Chart
Pie Chart
Correlation Matrix
Download filtered dataset
Key statistical insights

## Results

The dashboard successfully allows users to analyze datasets interactively. It provides visual insights, data cleaning options, and statistical summaries that help users understand patterns and trends in the data.

Any CSV dataset can be used with this dashboard.
